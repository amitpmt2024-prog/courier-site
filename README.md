# React + Vite App

Default entry: open `index.html` in the project root (Vite serves it).

A simple React application created with Vite.

## Setup

```bash
cd react-app
npm install
```

## Run

```bash
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```
